# 技术栈

- 使用**express**框架
- **Mysql**作为数据库存储数据
- **mysql2**插件与数据库链接
- **JWT**作为登录及整体验证方式
- **node-schedule**作为服务器执行定时任务插件

# 服务器

- 使用**Linux**系统
- 服务器配置需要**Nginx**、**Mysql**、**Node**环境
- **Node**版本**16.16.0**
